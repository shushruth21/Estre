# Shape Icons Directory

This directory contains the shape icons for L SHAPE, U SHAPE, and COMBO configurations.

## Current Files

✅ **l-sectionals.svg** - Icon for L-Sectionals (L SHAPE)
✅ **u-sectionals.svg** - Icon for U-Sectionals (U SHAPE)  
⚠️ **combo-modules.svg** - Icon for Combo Modules (COMBO) - Currently using U-sectionals as placeholder

## Image Specifications

- Format: SVG (vector format for crisp display at any size)
- ViewBox: Optimized for 61x46px viewBox
- Background: Transparent
- Style: Minimalist line drawings with gray outlines (#A3A2A1)

The images will be displayed at 64x64px (w-16 h-16) in the configurator UI with `object-contain` to maintain aspect ratio.
